آردوینو یک پلتفرم متن‌باز الکترونیکی است که بر پایه سخت‌افزار و نرم‌افزار انعطاف‌پذیر طراحی شده و به‌طور گسترده برای ساخت دستگاه‌های دیجیتالیِ قادر به دریافت اطلاعات از محیط و کنترل دنیای فیزیکی مورد استفاده قرار می‌گیرد. این دسته‌بندی شامل اکوسیستم کاملی از سخت‌افزارهای سازگار با Arduino است؛ از بردهای استاندارد میکروکنترلری مانند Uno و Mega گرفته تا ماژول‌های فشرده سری Beetle، به‌همراه انواع شیلدها و آرایه‌های سنسوری برای پایش محیطی یا داده‌های زیستی.
