# Arduino Mega 2560

برد Arduino Mega 2560 یکی از قدرتمندترین و کامل‌ترین بردهای خانواده Arduino است که برای پروژه‌های بزرگ‌تر و پیچیده‌تر طراحی شده است. این برد بر پایه میکروکنترلر ATmega2560 ساخته شده و نسبت به Arduino Uno و Nano دارای تعداد بسیار بیشتری پایه ورودی/خروجی، حافظه بیشتر و ارتباطات سریال متعدد است.

Arduino Mega معمولاً در پروژه‌هایی استفاده می‌شود که تعداد زیادی سنسور، نمایشگر، موتور یا ماژول به صورت هم‌زمان مورد استفاده قرار می‌گیرند.

---

# ویژگی‌های اصلی Arduino Mega

- تعداد بسیار زیاد پایه‌های ورودی/خروجی
- حافظه بیشتر نسبت به Uno و Nano
- دارای 4 پورت UART سخت‌افزاری
- مناسب پروژه‌های بزرگ و پیچیده
- سازگار با Arduino IDE
- پشتیبانی از انواع شیلدها و ماژول‌ها

---

# چرا از Arduino Mega استفاده کنیم؟

بردهای Arduino Uno و Nano برای بسیاری از پروژه‌ها کافی هستند، اما در پروژه‌های پیچیده‌تر معمولاً با محدودیت‌هایی مواجه می‌شویم:

- کمبود پایه دیجیتال
- کمبود حافظه
- تعداد محدود UART
- محدودیت در اتصال ماژول‌ها

Arduino Mega برای حل این مشکلات طراحی شده است.

---

# میکروکنترلر ATmega2560

قلب اصلی Arduino Mega میکروکنترلر AVR مدل ATmega2560 است.

این تراشه نسبت به ATmega328P بسیار قدرتمندتر بوده و امکانات بیشتری دارد.

---

## امکانات اصلی ATmega2560

- پردازنده 8 بیتی AVR
- حافظه Flash بزرگ
- چندین UART
- ADC داخلی
- PWM
- SPI
- I2C
- تایمرهای متعدد

---

# پایه‌های دیجیتال

Arduino Mega دارای 54 پایه دیجیتال است:

- D0 تا D53

این تعداد پایه برای پروژه‌های بزرگ بسیار مفید است.

---

# پایه‌های PWM

پایه‌های زیر قابلیت PWM دارند:

- D2 تا D13
- D44 تا D46

PWM برای:

- کنترل موتور
- تولید سیگنال
- کنترل نور
- کنترل سروو

کاربرد دارد.

---

# ورودی‌های آنالوگ

برد Mega دارای 16 ورودی آنالوگ است:

- A0 تا A15

این ویژگی باعث می‌شود بتوان تعداد زیادی سنسور آنالوگ را هم‌زمان متصل کرد.

ADC داخلی دارای دقت 10 بیت است.

---

# ارتباطات سریال

یکی از مهم‌ترین ویژگی‌های Arduino Mega وجود چندین UART سخت‌افزاری است.

---

# UART

برد Mega دارای 4 پورت UART است.

| UART | RX | TX |
|---|---|---|
| Serial | D0 | D1 |
| Serial1 | D19 | D18 |
| Serial2 | D17 | D16 |
| Serial3 | D15 | D14 |

این قابلیت برای پروژه‌هایی که چند ماژول سریال دارند بسیار مهم است.

---

# I2C

پایه‌های:

- SDA → D20
- SCL → D21

برای ارتباط I2C استفاده می‌شوند.

---

# SPI

پایه‌های:

- D50 → MISO
- D51 → MOSI
- D52 → SCK
- D53 → SS

برای ارتباط SPI استفاده می‌شوند.

---
