# Arduino Nano

برد Arduino Nano یکی از کوچک‌ترین و محبوب‌ترین بردهای خانواده Arduino است که بر پایه میکروکنترلر ATmega328P طراحی شده است. این برد تقریباً تمام قابلیت‌های Arduino Uno را در ابعادی بسیار کوچک‌تر ارائه می‌دهد و به همین دلیل در پروژه‌هایی که محدودیت فضا وجود دارد بسیار پرکاربرد است.

Arduino Nano به دلیل اندازه کوچک، مصرف توان مناسب، سازگاری گسترده با ماژول‌ها و امکان استفاده روی Breadboard، در میان دانشجویان، علاقه‌مندان الکترونیک، رباتیک و پروژه‌های اینترنت اشیاء محبوبیت زیادی دارد.

---

# ویژگی‌های اصلی Arduino Nano

- ابعاد کوچک و وزن کم
- سازگار با Arduino IDE
- مبتنی بر ATmega328P
- قابلیت نصب مستقیم روی Breadboard
- پشتیبانی از UART، SPI و I2C
- دارای 14 پایه دیجیتال و 8 ورودی آنالوگ
- مناسب پروژه‌های قابل حمل و فشرده

---

# تفاوت Arduino Nano و Arduino Uno

برد Nano از نظر عملکرد بسیار شبیه Uno است اما تفاوت‌های مهمی دارد.

| ویژگی | Arduino Nano | Arduino Uno |
|---|---|---|
| ابعاد | بسیار کوچک | بزرگ‌تر |
| نصب روی Breadboard | دارد | ندارد |
| نوع USB | Mini USB / Micro USB | USB Type-B |
| تعداد پایه آنالوگ | 8 | 6 |
| جک آداپتور | ندارد | دارد |
| مناسب پروژه‌های فشرده | بسیار مناسب | متوسط |

---

# نسخه‌های مختلف Arduino Nano

برد Nano در نسخه‌های مختلفی تولید شده است.

## Arduino Nano Classic

نسخه کلاسیک مبتنی بر ATmega328P که رایج‌ترین مدل Nano محسوب می‌شود.

---

## Arduino Nano Every

نسخه جدیدتر با پردازنده قوی‌تر و حافظه بیشتر.

---

## Arduino Nano CH340

بسیاری از بردهای Nano ارزان‌قیمت از چیپ CH340 برای ارتباط USB به سریال استفاده می‌کنند.

این مدل‌ها معمولاً نیاز به نصب درایور CH340 دارند.

---

| مشخصه | مقدار |
|---|---|
| میکروکنترلر | ATmega328P |

| EEPROM | 1KB |

---

# میکروکنترلر ATmega328P

قلب اصلی Arduino Nano میکروکنترلر AVR مدل ATmega328P است.

این تراشه شامل امکانات زیر است:

- پردازنده 8 بیتی AVR
- ADC داخلی
- PWM
- تایمرها
- UART
- SPI
- I2C

این پردازنده برای پروژه‌های سبک تا متوسط عملکرد مناسبی دارد.

---

# پایه‌های دیجیتال

Arduino Nano دارای 14 پایه دیجیتال از D0 تا D13 است.

این پایه‌ها می‌توانند به عنوان:

- ورودی دیجیتال
- خروجی دیجیتال

استفاده شوند.

---

# پایه‌های PWM

پایه‌های زیر قابلیت PWM دارند:

- D3
- D5
- D6
- D9
- D10
- D11

PWM معمولاً برای:

- کنترل سرعت موتور
- کنترل روشنایی LED
- تولید سیگنال

استفاده می‌شود.

---

# ورودی‌های آنالوگ

Nano دارای 8 ورودی آنالوگ است:

- A0 تا A7

در برخی نسخه‌ها، A6 و A7 فقط ورودی آنالوگ هستند و قابلیت دیجیتال ندارند.

ADC داخلی دارای دقت 10 بیت است.

---

# ارتباطات سریال

Arduino Nano از چندین پروتکل ارتباطی پشتیبانی می‌کند.

---

## UART

پایه‌های:

- D0 → RX
- D1 → TX

برای ارتباط سریال استفاده می‌شوند.

---

## I2C

پایه‌های:

- A4 → SDA
- A5 → SCL

برای I2C استفاده می‌شوند.

---

## SPI

پایه‌های:

- D10 → SS
- D11 → MOSI
- D12 → MISO
- D13 → SCK

برای ارتباط SPI استفاده می‌شوند.

---

# تغذیه برد

Arduino Nano را می‌توان از چند روش تغذیه کرد.

---

## USB

ساده‌ترین روش تغذیه و پروگرام برد.

---

## پایه VIN

برای اتصال منبع تغذیه خارجی استفاده می‌شود.

---

## پایه 5V

در صورت داشتن منبع تغذیه تنظیم‌شده می‌توان مستقیماً 5V به برد اعمال کرد.

---

# LEDهای روی برد

Nano دارای چند LED داخلی است:

| LED | کاربرد |
|---|---|
| ON | نمایش روشن بودن برد |
| TX/RX | ارتباط سریال |
| L | متصل به پایه D13 |

---

# پروگرام کردن Arduino Nano

برد Nano با Arduino IDE پروگرام می‌شود.

مراحل کلی:

1. اتصال برد با USB
2. نصب درایور در صورت نیاز
3. انتخاب Board
4. انتخاب Processor
5. انتخاب Port
6. Upload برنامه

---

# انتخاب Processor در Arduino IDE

در برخی بردهای Nano Clone باید Processor مناسب انتخاب شود.

معمولاً دو گزینه وجود دارد:

- ATmega328P
- ATmega328P (Old Bootloader)

اگر آپلود انجام نشد معمولاً باید Old Bootloader انتخاب شود.

# جمع‌بندی

Arduino Nano یکی از کاربردی‌ترین بردهای خانواده Arduino است که تقریباً تمام قابلیت‌های Uno را در ابعادی بسیار کوچک‌تر ارائه می‌دهد. این برد به دلیل اندازه کوچک، سادگی استفاده و سازگاری گسترده، انتخابی عالی برای پروژه‌های آموزشی، رباتیک و سیستم‌های قابل حمل محسوب می‌شود.

اگر به یک برد ساده، پایدار و کوچک برای پروژه‌های الکترونیکی نیاز دارید، Arduino Nano یکی از بهترین گزینه‌ها خواهد بود.
